# Top quark production from heavy ion collisions

## Installation

```
cmsrel CMSSW_8_0_28
cd CMSSW_8_0_28/src/
cmsenv
git clone git@github.com:pfs/TopFromHeavyIons.git UserCode/TopFromHeavyIons
scram b -j 9
cd UserCode/TopFromHeavyIons/
```
